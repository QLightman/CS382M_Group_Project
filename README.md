# CS382M_Group_Project
## Deep learning based approach

### How to run
#### Run the Unet model
```
cd deep_learning_based_methods
python Unet.py
```

#### Run the DenseUnet-v1 model
```
cd deep_learning_based_methods
python DenseUnet.py
```

#### Run the DenseUnet-v2 model
```
cd deep_learning_based_methods
python DenseUnet_v2.py
```
